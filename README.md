This repository contains a Power BI dashboard built using the cleaned Vehicle Sensor Dataset. The report visualizes key sensor metrics such as speed, acceleration, temperature, engine status, and battery performance. Interactive charts, KPI cards, and slicers provide clear insights into vehicle behavior and sensor activity patterns.
